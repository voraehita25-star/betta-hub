# 📦 product-info — ข้อมูลสินค้า (ที่พักก่อนเอาขึ้นเว็บ)

โฟลเดอร์นี้เก็บ "ข้อมูล + รูปต้นฉบับ" ของสินค้าแต่ละชิ้น แยกเป็น **1 โฟลเดอร์ต่อ 1 สินค้า**
ไฟล์ในนี้ **ไม่ขึ้นเว็บเอง** — เป็นแค่ที่พักข้อมูล (รูปที่ใช้จริงอยู่ใน `public/images/products/`)

## 📁 โครงสร้าง
```
product-info/
├── README.md            ← ไฟล์นี้ (สารบัญ)

├── betta-pellets/       อาหาร Hikari Betta Bio-Gold
│   ├── info.md          รายละเอียด + ลิงก์ + ราคา

└── betta-tank/          ตู้ปลานาโน 10 นิ้ว
```

## ➕ เพิ่มสินค้าใหม่
1. สร้างโฟลเดอร์ใหม่ (ตั้งชื่อตาม key เช่น `heater/`)
2. ก๊อป `_template.md` ไปไว้ในโฟลเดอร์นั้น ตั้งชื่อ `info.md` แล้วกรอกข้อมูล + วางรูป
3. บอก Claude ว่า "เพิ่มสินค้า <key> แล้ว" — Claude จะใส่ลิงก์ใน `affiliate.ts`, ย้ายรูปไป `public/`, และเขียนการ์ดในบทความให้

## หลักการ (จุดยืนเว็บ)
- รูป = รูปสินค้าจริงที่จะแนะนำ · จุดเด่นเขียนจากของจริง · ไม่ใส่ดาว/รีวิวปลอม
- เขียนราคาให้ชัดในไฟล์ `info.md` (Claude จะไม่เดาราคาเอง)
